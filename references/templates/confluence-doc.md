# Confluence Document Template

## Structure

1. **Overview** -- Brief summary and context
2. **Background** -- Why this document exists, what prompted it
3. **Details** -- The main content, organized with clear headers
4. **Action Items** -- Who needs to do what, with deadlines
5. **References** -- Links to related documents, tickets, or discussions

## Guidelines

- Use Confluence-friendly formatting (headers, tables, info/warning panels)
- Include @mentions for stakeholders where relevant
- Link to Jira tickets using ticket keys (e.g., DAT-123)
- Keep paragraphs short -- Confluence renders better with concise blocks
